    1. HTML [q] for Short Quotations

      2. Defines a short inline quotation

      WWF's goal is to: Build a future where people live in harmony with nature.

      [q]?[/q]

    2. HTML [blockquote] for Long Quotations

      3. Defines a section that is quoted from another source

      The HTML [blockquote] element defines a quoted section.

      For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.
      [blockquote cite="http://www.anyweb.com.html"] Element defines a quoted section [/blockquote]

    3. HTML [abbr] for Abbreviations

      4. Defines the title of a work

      The WHO was founded in 1948.

      Marking up abbreviations can give useful information to browsers, translation systems and search-engines.

      [abbr title="Long sentence"]LS[/abbr]

    4. HTML [address] for Contact Information

      5. Defines contact information for the author/owner of a document

      The HTML address element defines contact information (author/owner) of a document or article.

      Written by Jon Doe.
      Visit us at:
      Example.com
      Box 564, Disneyland
      USA

      [address>] Your address[br] [/address]

    5. HTML [bdo] for Bi-Directional Override

      6. Defines the text direction

      If your browser supports bi-directional override (bdo), the next line will be written from right to left (rtl):

      This line will be written from right to left

      [bdo dir="rtl"]This text will be written from right to left[/bdo]

    6. HTML [cite] for Work Title

      7. Defines the title of a work

      The HTML cite element defines the title of a work.

      Browsers usually display cite elements in italic.

      And this is me and Allah always generous by @nasser606060sad

      [cite]And this is me and Allah always generous[/cite]
